CAPCUT Easy CUT v1.1.0 - วิธีใช้งานแบบสั้น

สิ่งที่ผู้ใช้ต้องทำ
1. แตกไฟล์ ZIP นี้ก่อน
2. ดับเบิลคลิกไฟล์ RUN_CAPCUT_Easy_CUT.bat
3. รอให้ระบบตรวจและติดตั้ง Node.js, Python, ffmpeg และแพ็กเกจที่จำเป็น
4. เว็บจะเปิดเองที่ http://localhost:3000
5. ลากคลิปเข้าเว็บ แล้วใช้งานตามหน้าจอ

หมายเหตุ
- ต้องต่ออินเทอร์เน็ตในการเปิดครั้งแรก เพราะระบบต้องติดตั้งเครื่องมือและดาวน์โหลดโมเดลถอดเสียง
- ครั้งแรกที่ถอดเสียงอาจดาวน์โหลดโมเดลประมาณ 1.5GB
- ระหว่างใช้งานต้องเปิดหน้าต่างสีดำไว้ ถ้าปิดหน้าต่าง เว็บจะหยุด
- ถ้า Windows ถามสิทธิ์ติดตั้งโปรแกรม ให้กด Yes

โครงไฟล์ในชุดนี้
- RUN_CAPCUT_Easy_CUT.bat  = ไฟล์รันหลัก
- web                      = ไฟล์เว็บแอป CAPCUT Easy CUT
- HOW_TO_USE.txt           = วิธีใช้งาน

ถ้าเปิดไม่ได้
1. ตรวจว่าแตก ZIP ครบแล้ว ไม่ได้เปิดจากข้างใน ZIP โดยตรง
2. คลิกขวา RUN_CAPCUT_Easy_CUT.bat แล้วเลือก Run as administrator
3. ตรวจอินเทอร์เน็ต แล้วลองเปิดไฟล์รันอีกครั้ง
