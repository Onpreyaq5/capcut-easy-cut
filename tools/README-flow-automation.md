# ทำวิดีโอใน Google Flow แบบหลายช็อตอัตโนมัติ — โดยไม่ใช้ API

Google Flow (labs.google) **ไม่มี API สาธารณะ** ส่ง prompt วิ่งเข้าไปตรงๆ ไม่ได้
แต่ทั่วโลกมีวิธี "ป้อนผ่านหน้าเว็บ Flow ให้อัตโนมัติ" ซึ่ง**ไม่ต้องใช้ API** อยู่ 3 ระดับ:

| วิธี | ความง่าย | อัตโนมัติแค่ไหน | ความเสี่ยง |
|------|----------|-----------------|-----------|
| **A. คัดลอก+วางเอง** (ปุ่ม “เปิด Flow” ในเว็บ) | ง่ายสุด | ทำเองทีละช็อต | ไม่มี |
| **B. สคริปต์ของเรา** `flow-queue.user.js` | กลาง | วาง prompt ทีละช็อตให้ (กดปุ่มเดียว) | โปร่งใส อ่านโค้ดได้เอง |
| **C. Chrome Extension สำเร็จรูป** (Flow Automator ฯลฯ) | ง่าย | วาง+Generate+ดาวน์โหลดให้ครบ | ของบุคคลที่สาม เข้าถึงบัญชี Flow |

---

## วิธี B — สคริปต์ของเราเอง (แนะนำถ้าไม่อยากลงของคนอื่น)

1. ติดตั้งส่วนเสริม **Tampermonkey** (Chrome/Edge) หรือ **Violentmonkey**
   - Chrome Web Store → ค้น "Tampermonkey" → Add
2. เปิด Tampermonkey → **Create a new script** → ลบของเดิมทิ้ง → วางเนื้อไฟล์ `flow-queue.user.js` ทั้งหมด → **Save** (Ctrl+S)
3. เปิด https://labs.google/fx/tools/flow จะเห็นกล่อง **“🌾 ธัญกิจ · Flow Queue”** มุมขวาล่าง
4. ในเว็บธัญกิจ (ขั้นที่ 3) กด **“คัดลอก Bulk Prompts”** → กลับมาวางในกล่อง → กด **“โหลดคิว”**
5. ตั้งค่าใน Flow ครั้งเดียว: **Video · Omni Flash · 9:16 หรือ 16:9 · 10s · 1x**
6. กด **“วางช็อตถัดไป ▶”** ทีละช็อต (ติ๊ก “กด Generate อัตโนมัติ” ให้มันกดให้ก็ได้)
7. รอแต่ละคลิปเสร็จ แล้วดาวน์โหลด → ทำช็อตถัดไป

> ถ้าหาช่อง/ปุ่มไม่เจอ (Flow เปลี่ยนหน้า) ให้คลิกที่ช่องสร้างของ Flow ก่อน 1 ที แล้วลองใหม่

---

## วิธี C — Chrome Extension สำเร็จรูป (อัตโนมัติเต็มที่สุด)

ค้นใน **Chrome Web Store** คำว่า `google flow automation` / `veo automation` จะเจอหลายตัว เช่น
Flow Automator, Auto Flow, Veo Automation, FlowForge Pro, TurboFlow ฯลฯ

ขั้นตอนเหมือนกันหมด:
1. ติดตั้งเอ็กซ์เทนชัน (ดู permissions ให้ดีก่อน — มันเข้าถึงบัญชี Flow คุณ)
2. ในเว็บธัญกิจ กด **“คัดลอก Bulk Prompts”** (รูปแบบบล็อกละช็อต = ที่เอ็กซ์เทนชันพวกนี้รับ)
3. วางลิสต์ในเอ็กซ์เทนชัน → ตั้งค่า (Omni Flash, 10s, อัตราส่วน) → Start
4. มันจะป้อนเข้า Flow + กด Generate + โหลดวิดีโอให้เองทีละช็อต

> ⚠️ เอ็กซ์เทนชันเหล่านี้เป็นของบุคคลที่สาม บางตัวเสียเงิน บางตัวเก็บข้อมูล — เลือกตัวที่รีวิวดี/permission น้อย และเข้าใจว่าการ automate เว็บผู้ให้บริการอาจขัด ToS ของ Google ใช้ด้วยความรับผิดชอบของคุณเอง

---

## แล้ว API ต่างกันยังไง?
- **ไม่ใช้ API (A/B/C ข้างบน)** = ทำผ่านหน้าเว็บ Flow ของบัญชีคุณ ใช้เครดิต Flow ปกติ ไม่ต้องตั้งบัตร/คีย์
- **ใช้ API (Veo ผ่าน Gemini/Vertex)** = เว็บ gen วิดีโอเองได้เลย แต่ต้องเปิด billing ของ Google (ไม่มีฟรี) — เป็นคนละระบบกับหน้า Flow
